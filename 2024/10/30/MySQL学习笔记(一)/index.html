专有名词扩展

英语是了解计算机的重要工具

英语 中文（计算机行话，不涉及一般含义） 简写
database 数据库 DB
database management system 数据库管理系统 DBMS
data definition languages 数据定义语言 DDL
data manipulation languages 数据操作语言 DML
data control languages 数据控制语言 DCL
Data Query Language 数据查询语言，用来查询数据库中表的记录 DQL
alias 别名 as
structured query language 操作关系型数据库的编程语言 SQL

启动MySQL

打开方式有如下三种

通过命令行的方式打开(Windows11)

  1. 按win+r键，在弹出的窗口内输入cmd
  2. 再按ctrl+shift+enter以管理员的方式运行
  3. 输入net start MySQL服务名(例如本机是mysql80)
  4. 停止Mysql命令：net stop mysql80

  1. 按win+r键，在弹出的窗口内输入cmd
  2. 输入services.msc打开服务，找到对应的mysql，右键启动即可

按win+x键，选择“任务管理器”，在左侧栏点击“服务”，搜索mysql，然后鼠标右键点击“打开服务”(或者按照系统控制的方法打开服务也行)，然后找到mysql右键-“属性”，点开“启动类型”并选择自动，点击应用并确定即可实现开机自启动。

MySQL连接方式

对于第一次使用MySQL的读者，一开始是没有密码的，打开MySQL之后在mysql界面输入 ALTER USER ‘root’@‘localhost’ IDENTIFIED BY ‘自己设置的密码’;即可自行设置

方式 效果 难度 连接操作
MySQL 8.0 Command Line Client 和命令行一样，可视性不是特别强 低，安装MySQL自带的 开始界面搜索MySQL 8.0 Command Line Client，然后输入密码即可
Windows命令行 命令行，无需多言 中，需要在系统属性里的PATH里面配置环境变量(就是你的MySQL路径\bin) 打开命令行之后输入mysql -u用户名 -p密码 或者mysql -u root -p 加密输入密码防止泄露

SQL语言

无论是MySQL，还是oracle等其他的数据库，都遵守一定的SQL通用语法，特点如下

  1. 不区分大小写，但建议关键字大写
  2. 注释：#或者是--是MySQL特有的，/**/是多行注释
  3. 以分号结尾

增删查改数据

  1. 给指定字段添加数据

语法：INSERT INTO 表名 (字段名1, 字段名2, ...) VALUES (值1, 值2, ...);
示例

数据查询如下

如果是要给全部字段添加数据，就无需写字段名

  1. 批量添加

语法：INSERT INTO 表名 (字段名1, 字段名2, ...) VALUES (值1, 值2, ...), (值1, 值2, ...), (值1, 值2, ...) ;
示例

注意

  1. 插入数据时指定的字段顺序需要与值的顺序是一一对应的
  2. 字符串和日期类型要在引号中
  3. 数据应在规定范围内
  1. 修改数据

语法：UPDATE 表名 SET 字段名1 = 值1 , 字段名2 = 值2 , .... [ WHERE 条件 ] ;
示例

如果修改数据没有where语句，默认修改所有数据

语法：DELETE FROM 表名 [ WHERE 条件 ] ;
示例
